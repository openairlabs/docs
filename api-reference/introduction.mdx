# Flyweel API Introduction

<Callout type="info">
**Beta API Access** - The Flyweel API is currently in beta development. Contact our team for early access and integration support.
</Callout>

The **Flyweel API** provides programmatic access to your SpendOps platform data, enabling custom integrations, automated reporting, and advanced campaign analysis workflows. Built with modern REST principles, our API offers secure, scalable access to unified campaign performance data across Google Ads and Meta Ads.

## API Overview

### Core Capabilities

- **Campaign Data Access** - Retrieve performance metrics across all connected platforms
- **Advanced Filtering** - Use the same powerful filters available in the AdGrid dashboard
- **Real-time Metrics** - Access up-to-date campaign performance data
- **AI Insights** - Programmatic access to AI-generated optimization recommendations
- **Custom Reporting** - Build automated reporting solutions

### Beta Status

The Flyweel API is currently in beta, with limited access available to select partners and enterprise customers. During the beta phase:

- **Core endpoints** are available for campaign data retrieval
- **Authentication system** is fully functional and secure
- **Rate limiting** is in place to ensure platform stability
- **Documentation** is actively maintained and updated

## Getting Started

### Prerequisites

To use the Flyweel API, you need:
- An active Flyweel account with connected ad platforms
- API access permissions (contact support for beta access)
- Basic understanding of REST APIs and JSON

### Beta Access Request

To request API access during the beta period:

1. **Contact Support** - Email [api@flyweel.co](mailto:api@flyweel.co) with your use case
2. **Account Verification** - Confirm your active Flyweel subscription
3. **Access Approval** - Receive API credentials and documentation
4. **Integration Support** - Get dedicated support for your integration

## Base URL

```
https://api.flyweel.co/v1
```

## Authentication

The Flyweel API uses API key authentication. Include your API key in the request header:

```bash
Authorization: Bearer YOUR_API_KEY
```

## Rate Limiting

Current beta rate limits:
- **100 requests per minute** per API key
- **1,000 requests per hour** per API key
- Rate limit headers included in all responses
