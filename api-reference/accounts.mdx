# Accounts API

<Callout type="info">
**Account Management** - The Accounts API provides access to information about your connected Google Ads and Meta Ads accounts.
</Callout>

Retrieve details about connected advertising platforms and account status.

## Endpoints

### List All Accounts

```
GET /v1/accounts
```

### Get Account Details

```
GET /v1/accounts/{account_id}
```

## Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `platform` | string | Filter by platform: `google_ads`, `meta_ads` |
| `status` | string | Filter by status: `active`, `inactive` |

## Example Request

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" \
     "https://api.flyweel.co/v1/accounts"
```

## Account Status

- `active` - Account is connected and syncing data
- `inactive` - Account connection has been disabled
- `error` - Account has sync errors requiring attention
