# Campaigns API

<Callout type="info">
**Beta Endpoint** - The Campaigns API is available in beta. Contact support for access and integration assistance.
</Callout>

The Campaigns API provides access to unified campaign performance data across all your connected advertising platforms.

## Endpoint

```
GET /v1/campaigns
```

## Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `platform` | string | Filter by platform: `google_ads`, `meta_ads` |
| `status` | string | Filter by status: `active`, `paused`, `ended` |
| `date_from` | string | Start date (YYYY-MM-DD) |
| `date_to` | string | End date (YYYY-MM-DD) |
| `limit` | integer | Number of results (max 100) |
| `offset` | integer | Pagination offset |

## Example Request

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" \
     "https://api.flyweel.co/v1/campaigns?platform=google_ads&status=active&limit=10"
```
